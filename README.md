# Simple To-Do List App

A lightweight, easy-to-use to-do list web application built with HTML, CSS, and JavaScript.

## Features

- ✅ Add new tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Delete tasks
- ✅ Save tasks to your browser (tasks persist even after closing)
- ✅ Clean and simple user interface
- ✅ Works on desktop and mobile

## How to Use

1. **Clone or download** this project
2. **Open `index.html`** in your web browser
3. **Add a task** by typing in the input field and clicking "Add Task" (or press Enter)
4. **Mark complete** by clicking the checkbox next to a task
5. **Delete** a task by clicking the "Delete" button

No installation needed! Simply:
